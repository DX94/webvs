Class: ColorClip

Webvs. ColorClip

A component that clips colors to a different color depending on whether the source colors are above or below a reference color.

new ColorClip(options)

Parameters:
Name Type Description
options object

options object

Properties
Name Type Argument Default Description
mode string <optional>
 "BELOW"

comparison mode viz. BELOW, ABOVE, NEAR
color string <optional>
 "#202020"

reference color against which the the screen colors are compared
outColor string <optional>
 "#202020"

output color for clipped pixels
level number <optional>
 0

when mode is NEAR, this value decides the distance between source and reference colors below which pixels would be clipped. 0-1 normalized
