Class: FrameBufferManager

Webvs. FrameBufferManager

FrameBufferManager maintains a set of render targets and switches between them, when requested by different shader programs. Its used in EffectLists to compose rendering of the different Webvs.Component

new FrameBufferManager(width, height, gl, copier)

Parameters:
Name Type Description
width number

the width of the textures to be initialized
height number

the height of the textures to be initialized
gl WebGLRenderingContext

the webgl context to be used
copier Webvs.CopyProgram

an instance of a CopyProgram that should be used when a frame copyOver is required
